# Calculadora Profissional de Massas de Pizza — v4

Versão com ajustes de UX e processo:

- Campo renomeado para **Temperatura máxima do forno (°C)**.
- Resultado agora inclui:
  - Receita;
  - Percentuais do padeiro;
  - Processo;
  - Farinha e cocção;
  - **Modo de preparo recomendado**;
  - **Justificativa técnica**;
  - **Cronograma operacional sugerido**.
- O modo de preparo muda conforme estilo, forno, superfície, hidratação, fermentação e ingredientes compensadores.
- Mantém a separação entre **Napolitana clássica AVPN** e **Napolitana produzível**.
