# Pizzaiolo Pro — v6 Detroit integrada

Aplicativo estático HTML + CSS + JavaScript, pronto para GitHub Pages.

## Como publicar no GitHub Pages

1. Extraia o ZIP.
2. Suba o conteúdo extraído para a raiz do repositório, não o ZIP fechado.
3. O arquivo `index.html` deve ficar diretamente na raiz do repositório.
4. Em GitHub → Settings → Pages, escolha `Deploy from branch`, branch `main`, pasta `/root`.

## Novidades desta versão

- Menu com módulos **Calculadora profissional** e **Receitas de recheios**.
- Parte Detroit atualizada com base na planilha “Charlie Anderson Detroit Style Pizza Calculator V2”.
- Novos subestilos Detroit:
  - Detroit Artisan / Ultimate Charlie Anderson.
  - Detroit Traditional / Buddy's-style.
- Fórmula Detroit com:
  - 95% farinha branca forte;
  - 5% farinha integral;
  - 1% malte diastático;
  - 75% hidratação no perfil Artisan;

  - cálculo de queijo e molho por preferência;
