# Calculadora Profissional de Massas de Pizza v2

Versão alinhada à base metodológica pesquisada: estilos/subestilos, farinha parametrizada, fermento seco instantâneo, seco ativo, fresco e fermento natural/levain, horas equivalentes, superfícies térmicas e warnings técnicos.

## Como publicar

Suba todos os arquivos para um repositório GitHub e ative GitHub Pages em `Settings > Pages > Deploy from branch > main / root`.

## Fontes metodológicas principais

- Disciplinar AVPN 2024 para pizza napolitana.
- Associazione Pizza Romana para tonda, teglia e pala.
- Literatura técnica sobre alveógrafo, W, P/L, Falling Number, sourdough e transferência de calor.
- Perfis técnicos de uso de pedra, chapa de aço e formas.

## Aviso

A calculadora usa modelos práticos de engenharia de processo, não substitui ficha técnica real da farinha nem calibração empírica do seu forno.
