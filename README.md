# Calculadora Avançada de Massas de Pizza

Projeto estático em HTML, CSS e JavaScript puro, pronto para GitHub Pages.

## Como publicar

1. Extraia o ZIP.
2. Suba **os arquivos extraídos** para o repositório, garantindo que `index.html` fique na raiz.
3. No GitHub: Settings → Pages → Deploy from a branch → main → /root → Save.

## Arquivos principais

- `index.html`: interface
- `styles.css`: visual
- `app.js`: motor da calculadora

## Estilos suportados

- Napolitana clássica AVPN
- Napolitana produzível
- Neo-napolitana
- Romana
- Detroit
- Americana
- Brasileira
