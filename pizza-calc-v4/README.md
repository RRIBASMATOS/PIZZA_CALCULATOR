# Calculadora Profissional de Massas de Pizza — v4.1

Versão com motor térmico revisado.

## Ajustes desta versão

- O campo de forno representa a **Temperatura máxima do forno (°C)**.
- O assamento estimado agora considera a temperatura máxima informada, o estilo, o subestilo, o formato e a superfície.
- O modo de preparo recomendado evita termos subjetivos como “muito quente”.
- O resultado informa:
  - temperatura-alvo da pedra/chapa/forma;
  - configuração de forno para pré-aquecimento;
  - configuração de forno na entrada da pizza;
  - configuração de forno durante o assamento;
  - orientação de finalização com grill/broiler quando aplicável.

## Publicação

Suba todos os arquivos extraídos para o GitHub e ative GitHub Pages em:
