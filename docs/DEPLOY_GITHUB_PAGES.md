# Publicação no GitHub Pages

Esta versão foi preparada para GitHub Pages.

## Opção 1 — publicar pela raiz

1. Extraia o ZIP.
2. Suba o conteúdo extraído para a raiz do repositório.
3. Confirme que `index.html` aparece diretamente na primeira tela do repositório.
4. Vá em **Settings → Pages**.
5. Selecione:
   - Source: `Deploy from a branch`
   - Branch: `main`
   - Folder: `/ (root)`
6. Salve e aguarde alguns minutos.

## Opção 2 — publicar pela pasta docs

1. Extraia o ZIP.
2. Suba a pasta `docs/` para o repositório.
3. Vá em **Settings → Pages**.
4. Selecione:
   - Source: `Deploy from a branch`
   - Branch: `main`
   - Folder: `/docs`

## Checagem

Abra `gh-pages-check.html` no endereço do Pages. Se abrir, os arquivos estão sendo publicados corretamente.
