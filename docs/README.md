# Pizzaiolo Pro v10 — versão integrada

Esta versão mantém os módulos anteriores:

- Calculadora profissional
- Receitas de recheios
- Guia de preparo
- Curso gratuito

O módulo **Curso gratuito** incorpora todo o conteúdo da planilha `Formacao_Definitiva_Pizza_Artesanal.xlsx`, incluindo currículo, vídeos e links, bibliografia, experimentos, checklists, receitas base, plano de 32 semanas, dashboard e instruções de uso.

Os links da planilha foram extraídos dos hiperlinks reais do Excel, não apenas do texto visível “Abrir”.

Consulte `DEPLOY_GITHUB_PAGES.md` para publicar no GitHub Pages.
